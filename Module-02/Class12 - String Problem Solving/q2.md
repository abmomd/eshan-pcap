# 🔁 Palindrome Checker – Medium Level

## ⏱ Estimated Time
20–25 minutes

## 🎯 Objectives

- Practice **string manipulation**
- Learn how to **normalize text data**
- Use **loops and slicing**
- Apply logical thinking to solve real-world problems

---

## 📘 Scenario

A **palindrome** is a word, phrase, or sequence that reads the same **forward and backward**.

Examples:

```
"madam" → palindrome
"kayak" → palindrome
"loyal" → not a palindrome
```

However, in real-world scenarios, palindrome checks are often more flexible.

---

## 🛠 Your Task

Write a Python program that:

1. Asks the user to enter some text  
2. Checks whether the text is a palindrome  
3. Prints the result  

---

## ⚙️ Rules

Your program must follow these rules:

### 🔹 Case Insensitivity

- Treat uppercase and lowercase letters as the same

```
"Madam" → palindrome
```

---

### 🔹 Ignore Spaces

- Spaces should NOT affect the result

```
"nurses run" → palindrome
```

---

### 🔹 Empty String Rule

- An empty string is **NOT a palindrome**

---

## 🧪 Sample Input 1

```
Ten animals I slam in a net
```

### ✅ Output

```
It's a palindrome
```

---

## 🧪 Sample Input 2

```
Eleven animals I slam in a net
```

### ❌ Output

```
It's not a palindrome
```

---

## 💡 Hints

- Use:
  ```
  lower()
  replace()
  ```
- Remove spaces before checking
- Compare string with its reverse:
  ```
  text == text[::-1]
  ```

---

## 🚨 Edge Cases

- Input: `" "` (only spaces) → not a palindrome
- Input: `"A"` → palindrome
- Input with punctuation (optional handling)
