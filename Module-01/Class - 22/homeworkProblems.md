# 🚀 Problem 1: Space Mission Control Console

## 📘 Story

You are developing software for a space agency. Before every rocket launch, engineers need a console that performs system checks and mission preparations.

---

## 🎯 Your Task

Create a program that:

1. Imports the required modules appropriately.
2. Displays all available functions in the `math` module.
3. Calculates:
   - Escape velocity approximation using `sqrt()`
   - Fuel tank volume using `pi`
4. Randomly select one launch pad from:
   - Pad A
   - Pad B
   - Pad C
   - Pad D
5. Generate a random mission ID (10000–99999).
6. Display:
   - Operating System
   - Processor
   - Python Version

---

## Concepts

- import
- dir()
- math
- random
- platform
