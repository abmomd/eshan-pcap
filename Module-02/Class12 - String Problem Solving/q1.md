25–30 minutes

- Strengthen skills in **string manipulation**
- Practice working with **loops and conditions**
- Use **ASCII/Unicode transformations (`ord()` and `chr()`)**
- Handle **user input validation**
- Implement a real-world **encryption algorithm**

---

## 📘 Scenario

You are already familiar with the **Caesar Cipher**, a simple encryption technique.

In the basic version:
- Each letter is shifted by **1 position**
- Example:
  ```
  a → b
  z → a
  ```

Now, you will build an **enhanced version** of this cipher.

---

## 🛠 Your Task

Write a Python program that:

1. Asks the user to enter a line of text  
2. Asks the user to enter a **shift value (integer from 1 to 25)**  
3. Encrypts the text using the Caesar cipher  
4. Prints the encrypted result  

---

## ⚙️ Rules

### 🔹 1. Shift Value

- Must be an integer in range:
  ```
  1 to 25 (inclusive)
  ```
- If invalid input is entered:
  - Keep asking the user until valid input is provided

---

### 🔹 2. Character Handling

- **Alphabetical characters**:
  - Shift them forward by the given value
  - Wrap around if needed:
    ```
    z → a
    Z → A
    ```

- **Preserve case**:
  - Lowercase stays lowercase
  - Uppercase stays uppercase

- **Non-alphabetical characters**:
  - Must remain unchanged  
  (e.g., spaces, numbers, punctuation)

---
